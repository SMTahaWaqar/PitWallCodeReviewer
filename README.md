An F1-inspired code review tool that turns raw code into structured, actionable feedback.

## What it does
Paste a code snippet and receive feedback across:
- Strategy (structure & design)
- Reliability (bugs & edge cases)
- Performance
- Maintainability
- Race control flags

## Why
Built to practice thinking in abstractions and structured code, and to explore AI-assisted developer tooling.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS v4
- OpenAI Responses API (structured outputs)
- Vercel
